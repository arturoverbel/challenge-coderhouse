# E-commerce Project
This is a e-commerce project. Made with React JS. Challenge for coderhouse

## Features
- NavBar with categories

## Tech Stack
- React
- Tailwind CSS
- Vite

## How to use
1. Clone the repo
``` bash
git clone https://github.com/arturoverbel/challenge-coderhouse
```

2. Install dependencies
``` bash
npm install
```

3. Start the dev server
``` bash
npm run dev
```
